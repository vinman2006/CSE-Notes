## ✦ Minimal. Fast. Academic.

**Vinman Notes** is a highly polished, responsive web application designed for students. It prioritizes a fluid, minimalist UI/UX, bringing a premium feel to academic resources. The interface strictly adheres to a monochromatic design language, ensuring focus remains entirely on the content.

It provides an **Excalidraw-inspired** experience—clean, wireframe-like, and highly responsive.

<br />

## ❖ Key Features

- **Monochrome Design System** — A strict black-and-white aesthetic that eliminates distractions and reduces cognitive load.
- **Priority Structuring** — The subject grid intelligently prioritizes lab subjects over theory subjects to align with student workflows.
- **Fluid & performant** — High-performance, physics-based entry animations and interactions powered by **GSAP** and **Framer Motion**.
- **Smooth Scrolling** — Integrated **Lenis** for a buttery-smooth narrative scroll experience.
- **Subtle 3D Interactions** — Cinematic, lightweight 3D elements powered by **React Three Fiber**.
- **No Fluff** — Removes redundant hero elements to drop users directly into the action.

## ⚙️ Development

To run the platform locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vinman2006/CSE-Notes.git
   cd vinmannotes.online
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

<br />

## ⌘ Design Philosophy

The portal is designed to feel like a high-end, interactive document. By relying on a stark grayscale palette, the interface demands high-quality typography, precise spacing, and intentional motion to convey hierarchy.
